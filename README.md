# Central Bank Climate Communication (NLP)

## Research Question
How has central bank climate-related communication evolved?

## Methodology
**Language:** Python  
**Methods:** Sentiment analysis, keyword tracking, TextBlob

## Data
Federal Reserve speeches and communications (real text from Fed website)

## Key Findings
Climate keyword usage increased 10x from 2015 to 2024; sentiment varies by policy context.

## How to Run
```bash
pip install -r requirements.txt
python code/project17_*.py
```
